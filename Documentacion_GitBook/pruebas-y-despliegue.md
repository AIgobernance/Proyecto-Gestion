# Pruebas y despliegue

**Pruebas a implementar:**

* Unitarias (PHPUnit) en Laravel.
* Integración (React Testing Library + Jest).
* Validación de flujos API ↔ n8n ↔ IA.
* Envío real de correo/SMS en entorno de staging.

**Despliegue:**

* **Frontend:** React (Vite) → Netlify o Vercel.
* **Backend:** Laravel (PHP 8.2) → Servidor Apache/Nginx.
* **Base de datos:** SQL Server.
* **n8n:** Docker o servidor dedicado (Webhook público).
